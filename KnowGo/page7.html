Математика — Многочлены

Определение

Многочлен - это сумма одночленов.

Например: 12x2b3 - 5a2, -4xy2 + a - 3

Члены многочлена - одночлены, входящие в его состав.

Например: Многочлен -4xy2 + a - 3 состоит из трех членов: -4xy2, a и -3

Одночлен можно считать многочленом, состоящим из одного члена. Многочлен, содержащий два члена, называется двучленом. Многочлен, содержащий три члена, называется трёхчленом. Если многочлен имеет более трех слагаемых, то он называется просто многочленом.

Действия с многочленами

Сложение и вычитание

При сложении и вычитании:

  • Раскрываем скобки;
  • Приводим подобные члены полученного многочлена.

    • Пример: (-7x2 + 2xy + 3y2) + (-4y2 - 2xy + 3x2) = -7x2 + 2xy + 3y2 - 4y2 + -2xy + 3x2 = -4x2 - y2,

    (-6x2 - 5x + 7) - (-6x2 - 4x - 8) = -6x2 - 5x + 7 + 6x2 + 4x + 8 = -x + 15

    Умножение многочлена на одночлен

    Для умножения одночлена на многочлен умножаем одночлен на каждый член многочлена и находим сумму результатов.

    Пример: -3a2b • 2ab3 = (3 • 2) • a2+1 • b1+3 = 6a3b4

    Умножение многочлена на многочлен

    Для нахождения произведения многочленов умножаем каждый член первого многочлена на каждый член второго многочлена и находим сумму результатов.

    Пример: (a + b) • (c + d) = a • c + a • d + b • c + b • d = ac + ad + bc + bd

    Формулы сокращенного умножения

  • Квадрат суммы (a + b)2 = a2 + 2ab + b2

    • Чтобы сумму возвести в квадрат, можно к сумме квадратов первого и второго выражений прибавить удвоенное их произведение.

    Пример: (z + 5)2 = z2 + 2 • z • 5 + 52 = z2 + 10z + 25

  • Квадрат разности (a - b)2 = a2 - 2ab + b2

    • Чтобы разность возвести в квадрат, можно от суммы квадратов первого и второго выражений вычесть удвоенное их произведение.

    Пример: (z - 5)2 = z2 - 2 • z • 5 + 52 = z2 - 10z + 25

  • Разность квадратов (a - b)(a + b) = a2 - b2

    • Чтобы перемножить сумму и разность двух выражений, можно каждое выражение возвести в квадрат и найти их разность.

    Пример: (z - 5)(z + 5) = z2 - 52 = z2 - 25

    Деление многочлена на одночлен

    Чтобы многочлен разделить на одночлен, можно каждый член многочлена разделить на данный одночлен и найти сумму результатов.

    Пример:

    (45x7y + 27x6z) : 9 = 45x7y : 9 + 27x6z : 9 = 5x7y + 3x6z.
