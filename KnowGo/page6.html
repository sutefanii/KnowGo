Математика — Неравенства

Числовые промежутки

Так как у неравенств бесконечное множество решений, записать все невозможно. Решение можно отобразить на оси координат. Отображённое на оси координат решение неравенства можно записать в виде числового промежутка.

Вид неравенства и обозначение точки на оси координат Запись принадлежности конечной точки промежутка
≤ или ≥ (конечная точка включена) [ или ] - квадратные скобки
< или > (конечная точка не включена) ( или ) - круглые скобки

Неравенства одинакового смысла, двойные неравенства

Неравенства вида a > b и c > d или a < b и c < d, называют неравенствами одинакового смысла.

Неравенства вида a > b и c < d или a < b и c > d, называют неравенствами противоположного смысла.

Например:

  • Неравенства x > -5 и y > 17 являются неравенствами одинакового смысла.
  • Неравенства x < -5 и y > 17 являются неравенствами противоположного смысла.

    • Основные свойства числовых неравенств

    Cвойство 1:

  • Если a > b, a b > c, то а > c

    • Свойство 2:

  • Если a > b, то a + c > a + b

    • Свойство 3:

  • Если a > b и k > 0, то ak > bk

    • Линейные неравенства

    Линейным неравенством называется неравенство, которое дано или преобразуемо в форму:

  • ax > b
  • ax < b
  • ax ≥ b
  • ax ≤ b,

    • где a, b - числа и x - переменная.

    Пример:

    a - 5 > 0;
    a > 5;
    Ответ: a ∈ (5; +∞)

    Что значит «решить линейное неравенство»

    Решением неравенства с переменной называют значение переменной, которое обращает неравенство с переменной в верное числовое неравенство.

    Решить неравенство - это значит найти все его решения или доказать, что их нет.

    Пример:

    Являются ли решением неравенства 3x < 5 значения x, равные:
    a) -1
    б) 3?

    Решение.

    a) Вместо x в неравенство подставим число -1:

    3 × (-1) < 5;
    -3 < 5
    -3 < 5 - верно, поэтому x = -1 - решение неравенства

    б) Вместо x подставим число 3:

    3 × 3 < 5;
    9 < 5
    9 < 5 - не верно, поэтому x = 3 не является решением.

    Квадратные неравенства

    Обший вид квадратных неравенств - это ax2 + bx + c > 0 ( < 0, ≤ 0, ≥ 0 )

    Множество решений квадратного неравенства легкко определить, приблизительно начертив график функции y = ax2 + bx + c (параболу)


    Шаги решения квадратного неравенства.

    1. Определяются точки пересечения параболы и оси x с помощью уравнения ax2 + bx + c = 0.

    Вспомним формулы корней квадратного уравнения:

    D = b2 - 4ac
    x1 = -b + D 2a , x2 = -b - D 2a .

    2. Учитывая количество корней и знак коэффициента a, чертится график параболы.

    3. Выбираются пустые или закрашенные точки, в зависимости от вида знака неравенства:
    ●, если стоит знак нестрогого неравенства - ≤ или ≥;
    ○, если стоит знак строгого неравенства - < или >.

    4. Закрашивается правильный интервал

    5. Записывается ответ.


    Пример:

    Решить квадратное неравенство -2x² + 4x - 5 ≤ 0

    Решение:

    -2x² + 4x - 5 ≤ 0 | × (-1)
    2x² - 4x + 5 ≥ 0;
    D = 16 - 4 × 2 × 5 = -24;

    Ответ:
    x ∈ ( -∞; +∞ ), или x ∈ ℝ
